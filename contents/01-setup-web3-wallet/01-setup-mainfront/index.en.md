* Wallet source clone in your cloud9

   ```shell
   git clone git@ssh.gitlab.aws.dev:blockchain-tfc/web3-wallet-workshop.git
   ```

* Dependency install

   ```shell
   cd main-front
   npm install
   ```

* Launch main front

   ```shell
   npm run start
   ```
